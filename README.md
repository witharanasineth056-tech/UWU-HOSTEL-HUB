UWU Hostel Hub is a web-based hostel finder platform developed for students of Uva Wellassa University of Sri Lanka. The website helps students easily discover hostels located near the university and compare different accommodation options. It provides a simple, responsive, and user-friendly interface that works on both desktop and mobile devices. The project was developed using HTML, CSS, and JavaScript, making it lightweight, fast, and easy to use.

The website includes several useful features to improve the hostel searching experience. Users can search and filter hostels based on their preferences, such as hostel type, monthly rent, distance, and available facilities. Each hostel has a detailed page with information including room prices, amenities, images, and contact details. The platform also allows users to create an account, save their favourite hostels for later, and quickly access important information through an organized and attractive interface. These features help students find suitable accommodation more efficiently and conveniently.
